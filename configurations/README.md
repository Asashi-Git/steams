# Project Configuration

## Overview

This directory contains the **original (raw) configuration files** for both the Apache HTTP Server and the MariaDB database.

These files are provided as-is, without additional explanations or modifications, and reflect the exact configurations used in the environment.

## Directory Structure

- **apache/**
    
    - Contains the raw Apache configuration file(s)
    - Example: `httpd.conf`
- **mariadb/**
    
    - Contains the raw MariaDB configuration file(s)
    - Example: `steams_db.sql`

## Purpose

These subfolders are intended for:

- Direct inspection of the **actual server configurations**
- Troubleshooting or auditing setup details
- Comparing raw configurations with the documented versions

## Notes

- No abstraction or documentation is included inside these files
- For detailed explanations, refer to the documented `.md` files in the corresponding documentation directories
